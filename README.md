## Overview

This project is a fully responsive product page + mini cart component built with HTML5, CSS3 (Grid + Flexbox), and Vanilla JavaScript, without using any third-party frameworks.  
It features persistent cart state via LocalStorage, variant selection, image gallery, keyboard navigation, and accessible modal/drawer UI.

| Hosting | GitHub Pages |

---

## Features

- Product gallery with thumbnail navigation and keyboard controls (← / →)
- Variant selection (color and size)
- Mini cart drawer accessible via cart button, includes add/remove, quantity, subtotal
- Cart state saved in LocalStorage (persists after page refresh)
- Accessibility focused: semantic tags, skip link, keyboard navigation, focus rings
- Fully responsive from mobile (320px) up to desktop (1440px)
- Clean codebase, no frameworks or dependencies
